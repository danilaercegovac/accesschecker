# Accessibility Checker API

## Описание

Accessibility Checker — это API-сервис для проверки веб-страниц на соответствие стандартам доступности (accessibility). Сервис анализирует HTML-код страницы и генерирует отчёт в формате PDF с результатами проверки.

## Требования
* Node.js версии 16.x или выше
* npm или yarn
* Права доступа к директории проекта

## Инструкция по установке
1. Клонируйте репозиторий:

`git clone https://github.com/danilaercegovac/accesschecker.git`

`cd accessibility-checker`

2. Установите зависимости:

`npm install`

3. Запустите сервер:

`npm install`
